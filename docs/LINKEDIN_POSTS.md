## Option 4: Technical Deep-Dive

```
🔬 Deep Learning for Materials Engineering: A Technical Overview

I've just open-sourced Material AI - an ensemble ML system for TIG weld property prediction. Here's what's under the hood:

Architecture:
• LightGBM: Fast gradient boosting for tabular data
• FT-Transformer: Feature tokenization with attention mechanisms
• CVAE: Conditional variational autoencoder for uncertainty quantification
• Weighted ensemble with physics constraints

Technical Highlights:
✓ 16 input features (welding parameters, thermal history, geometry)
✓ 3 output properties (yield, UTS, elongation)
✓ Ramberg-Osgood curve generation
✓ SHAP explainability
✓ Physics-aware constraints (Yield < UTS)

Performance Metrics:
• Yield Strength R²: 0.655
• UTS R²: 0.631
• Elongation R²: 0.781
• Mean latency: 18.4ms
• Throughput: 54.3 pred/sec

Production Features:
• FastAPI REST API with OpenAPI docs
• Streamlit GUI with 3 input modes
• Docker containerization
• Prometheus metrics

Python 3.8+ | PyTorch | LightGBM | Scikit-learn | Streamlit | FastAPI | Docker | Pytest
```
